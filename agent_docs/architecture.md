## Data Flow (detailed)

```
src/data.ts
  └─ COMPOSABLE_COW_DEPLOYMENTS  {address, startBlock} per chain
  └─ ComposableCowContract        {abi, chain: {mainnet: ...}}

ponder.config.ts
  └─ chains: { mainnet: { id: 1, rpc: MAINNET_RPC_URL } }
  └─ contracts: { ComposableCow: ComposableCowContract }
       │
       │  Ponder runtime syncs blocks, emits events
       ▼
src/application/handlers/composableCow.ts
  └─ ponder.on("ComposableCow:ConditionalOrderCreated", async ({ event, context }) => {
       1. Destructure event.args.params  →  { handler, salt, staticInput }
       2. encodeAbiParameters(tuple) + keccak256  →  hash
       3. context.db.insert(transaction).values({...}).onConflictDoNothing()
       4. context.db.insert(conditionalOrderGenerator).values({...}).onConflictDoNothing()
     })
       │
       ▼
schema/tables.ts  →  transaction table (hash, chainId, blockNumber, blockTimestamp)
                 →  conditionalOrderGenerator table (eventId, chainId, owner, handler, salt, staticInput, hash, orderType, status, decodedParams, txHash)
                 →  discreteOrder table (orderUid, chainId, conditionalOrderGeneratorId)  [stub for M3]

src/api/index.ts
  └─ Hono app
       app.use("/sql/*", client({ db, schema }))   // Ponder SQL client
       app.use("/",       graphql({ db, schema }))  // GraphQL
       app.use("/graphql", graphql({ db, schema }))
```

## Virtual Modules (Ponder codegen)

`ponder-env.d.ts` is auto-generated by `ponder codegen` and provides types for:
- `ponder:registry` → `ponder` (event registration)
- `ponder:schema` → table references for handlers and API
- `ponder:api` → `db` for the API layer

Re-run `pnpm codegen` any time `ponder.config.ts` or `ponder.schema.ts` changes.

## Multi-Chain Readiness

M1 wires mainnet only. To add a chain:

1. `src/data.ts` — add entry to `COMPOSABLE_COW_DEPLOYMENTS` and `ComposableCowContract.chain`
2. `ponder.config.ts` — add entry to `chains` with `id` and `rpc` env var
3. `.env.local.example` — uncomment the `_RPC_URL` for the new chain

Future chains: gnosis (COW-7xx), arbitrum (COW-7xx).
